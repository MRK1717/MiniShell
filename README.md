# MiniShell

## Prezentare generală
MiniShell este o implementare simplificată a unui shell Unix, realizată în C ca proiect de facultate.
Proiectul oferă funcționalități de bază precum executarea de comenzi, redirecționarea input/output și pipe-uri.

## Funcționalități
- Execuție de programe externe
- Comenzi built-in (ex.: `cd`, `exit`)
- Suport pentru pipe-uri și redirecționare
- Parsare simplificată a liniei de comandă

## Cerințe
- Compilator C (de ex. GCC)
- Sistem de operare compatibil POSIX (Linux, Unix, macOS)

## Compilare
Utilizează Makefile-ul inclus:
